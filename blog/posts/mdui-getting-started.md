MDUI 2 is a Material You component library based on Web Components.

## Quick Start

Add these to your `<head>`:

```html
<link rel="stylesheet" href="https://unpkg.com/mdui@2/mdui.css" />
<script src="https://unpkg.com/mdui@2/mdui.global.js"></script>
```

## Components

MDUI provides many components:

- `<mdui-button>` — buttons
- `<mdui-card>` — cards
- `<mdui-chip>` — chips
- `<mdui-list>` — lists
- `<mdui-text-field>` — text inputs
- `<mdui-layout>` — layout system
- `<mdui-top-app-bar>` — app bars
- `<mdui-navigation-drawer>` — side navigation
- `<mdui-avatar>` — avatars
- `<mdui-snackbar>` — snackbars / toasts
