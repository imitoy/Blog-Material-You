This is a standalone blog project based on Material You design, powered by MDUI 2 (Material Design 3).

The original concept was from a Material Design 3 theme. This version reimplements all functionality without any framework dependency — using OpenResty, Lua, MariaDB, and MDUI Web Components.

All posts are written in Markdown with front matter. Comments are stored in MariaDB.
